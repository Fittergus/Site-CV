JavaScript

Le langage qui rend le web interactif

Qu'est ce que le JavaScript ?

JavaScript est un langage de programmation qui permet de rendre les pages web interactives. Il est utilisé pour créer des effets dynamiques, des animations et des fonctionnalités avancées sur les sites web.

Exemples de code

                    
  document.querySelector('button').addEventListener('click', () => {
  alert('Hello JS!');
  });
